----------Weather App in Python----------

National Public School, Indiranagar
Grade XI-B

Computer Science Project: Weather Application
Manu Dhulipala, Sanjay Shriram, Aditi Kamat

A modern and functional weather app with the use of OpenWeather API for weather data and python gui library customtkinter.

Language: Python
Software used: OpenWeatherMap API services
Packages and Libraries: customtkinter, PIL (Python image library), datetime, requests, ctypes (windll, etc.)
